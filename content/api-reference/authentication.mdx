<h2 id="h-header">Required header</h2>

```text
Authorization: Bearer nse_live_01HKBZ...
```


<h2 id="h-envs">Environments</h2>

<table>
  <tr><th>HTTP</th><th>Code</th><th>Meaning</th></tr>

  <tr><td>401</td><td>`invalid_auth`</td><td>Key not recognised, revoked, or expired.</td></tr>
  <tr><td>401</td><td>`environment_mismatch`</td><td>Sandbox key against a production URL, or the reverse.</td></tr>
  <tr><td>429</td><td>`rate_limited`</td><td>See <a href="/developer/rate-limits">Rate limits</a>.</td></tr>
</table>
